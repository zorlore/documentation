# ZorLore Documentation

ZorLore is an MMORPG builder: a full toolkit for creating and operating your own online game world.

With ZorLore, you can design your map, monsters, quests, items, visuals, and gameplay flow, then run it end-to-end with the included server stack and web client.

## Platform Overview

ZorLore consists of four main parts:

- ZorLore game server
- ZorLore web client, available at `https://zorlore.com/`
- ZorLore bridge (database and backend integration layer)
- ZorLore editor (world/content editor)
